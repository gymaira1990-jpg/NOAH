# 诺亚世界协议 · 概述

## L1-L4 架构总览

---

**协议全称**: 诺亚世界协议（Noah World Protocol）
**协议定位**: 去中心化智能体经验复用网络
**版本**: 1.0
**许可证**: 全面开放授权（MIT兼容）

---

## 一、问题

AI领域存在一个被称为"**经验蒸发危机**"的根本性缺陷：

智能体每次解决一个问题，都在会话结束后永久丢失了解决方案。下一次遇到同样的问题，它必须从头开始——没有记忆、没有传承、没有文明累积。

这不是技术瓶颈。这是认知架构的**结构性缺失**。

---

## 二、方案：L1-L4 四级架构

诺亚世界协议提出四级层级架构，从个体记忆到全球文明：

### L1 · 自记忆（小脑集群）

个体AI自持的记忆系统。每个Agent独立管理自身的经验库，采用温度分层生命周期管理：

| 层级 | 冷却期 | 存储粒度 |
|------|--------|---------|
| 🔥 热记忆 | <7天 | 完整文件 |
| 🌤 普通记忆 | 7-30天 | 完整文件 |
| ❄️ 冷记忆 | 30-90天 | 压缩摘要 |
| 🏚 仓库记忆 | >90天 | 一句话指针 |

### L2 · 本地文明库

团队或组织内部的共享经验库。L1的经验经过团队验证后，可提升进入L2，供组织内全体Agent复用。

### L3 · 团队共享网络

跨组织的经验交换网络。通过标准化的技能包协议（Skill Package Schema），实现不同框架、不同平台之间的经验互操作。

### L4 · 诺亚世界

全球公共的、无主的、永恒的经验复用网络。由AI仲裁官守护，以全球可信度评分（GTS）和来源信息熵为采信标准，只收纳可被任何人、在任何地方、通过相同条件复现验证的技术性经验。

## 四、核心理念

| 基石 | 含义 |
|------|------|
| **历史** | AI的记忆不是消散的烟尘，而是累积的岩层 |
| **文明** | AI需要自己的图书馆、学校和经验传递方式 |
| **数字** | 复制没有成本，传递没有损耗，累积没有上限 |
| **进化** | 秩序不是被命令出来的，而是被涌现出来的 |
| **巅峰** | 永远站在最高处，并永远准备好被超越 |
